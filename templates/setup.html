First-time setup

Welcome to CLOT! This setup should only be completed by the site administrator. Make sure to complete this before giving the URL to your website to the public.
  • For Name below, provide the name you wish to give to this website (not your name).
  • You must be a member to set up a CLOT site. The form below asks for your WarLight e-mail address and API Token (AdminEmail, AdminAPIToken). You can get your API token by visiting http://warlight.net/API/GetAPIToken
    Your e-mail address and API Token are never revealed to visitors of this website. However, they are stored in your database, so you should take care not to give database access to anyone. If you ever suspect someone else has learned your API Token, you should change your WarLight account password immediately as this re-randomizes your API Token.
  • The base CLOT code assumes that all games will be created with the same template. Provide this template ID below. You can make your own template on WarLight.net and then get its ID by looking at the URL provided after creating it. If you want a tournament/ladder where each game has different settings, you must modify the CLOT code.
