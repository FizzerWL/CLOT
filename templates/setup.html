First-time setup

Welcome to CLOT! This setup should only be completed by the site administrator. Make sure to complete this before revealing the URL of your website to the public.
  • Below, you will give your CLOT a name. This will be displayed at the top of your website.
  • You must be a WarLight member to set up a CLOT site. The form below asks for your WarLight e-mail address and API Token. You can get your API token by visiting http://warlight.net/API/GetAPIToken. If this link says "No auth found", you must first sign into your WarLight.net account.
    Your e-mail address and API Token are never revealed to visitors of this website. However, they are stored in your database, so you should take care not to give database access to anyone. If you ever suspect someone else has learned your API Token, you should change your WarLight account password immediately as this re-randomizes your API Token.
  • Below, you can choose whether you want to allow non-members to join your CLOT. If you allow non-members in, you will not be able to use any member-specific features, such as custom scenarios or adjusting the luck percentage.
